## 광고 배치가 수익을 결정한다

같은 트래픽이라도 광고 배치에 따라 수익이 2-3배 차이가 납니다. RPM이 낮다면 콘텐츠 품질보다 광고 배치를 먼저 점검하세요.

실제로 광고 위치만 바꿔서 월 수익이 78,000원에서 142,000원으로 올라간 사례를 분석했습니다.

## 핵심 원칙: 뷰포트(화면) 안에 광고를 배치하라

가장 중요한 원칙입니다. **사용자가 스크롤 없이 볼 수 있는 영역**에 광고가 있어야 클릭 가능성이 높아집니다.

### 클릭률이 높은 위치 TOP 5

1. **본문 상단** — 제목 바로 아래, 첫 문단 전
2. **본문 중간** — 스크롤 50% 지점
3. **사이드바 상단** — PC에서 화면 고정(sticky) 광고
4. **본문 하단** — 콘텐츠 끝, 댓글 전
5. **인콘텐츠** — 본문 중간에 삽입되는 자동 광고

## 광고 크기별 성능 비교

| 광고 크기 | CTR | RPM | 추천 위치 |
|-----------|-----|-----|-----------|
| 반응형 광고 | 높음 | 높음 | 모든 위치 기본값 |
| 300×250 (중간 직사각형) | 중간 | 높음 | 사이드바, 본문 중간 |
| 728×90 (리더보드) | 낮음 | 중간 | 헤더, 푸터 |
| 160×600 (와이드 스카이스크래퍼) | 중간 | 중간 | 사이드바 |
| 320×100 (대형 모바일 배너) | 높음 | 높음 | 모바일 하단 |

> **결론**: 반응형 광고를 기본으로 사용하되, 사이드바에는 300×250을 고정 배치하세요.

## PC 최적 배치 전략

### 기본 구조

```
[헤더 / 네비게이션]
[제목]
[광고 1 - 728×90 또는 반응형]  ← 본문 상단
[도입 문단 1-2개]
[광고 2 - 300×250 인콘텐츠]   ← 본문 1/3 지점
[본문 중반]
[광고 3 - 반응형]               ← 본문 2/3 지점
[마무리 문단]
[광고 4 - 728×90]              ← 본문 하단
[관련 글]
```

### 사이드바 배치

```
[사이드바]
├─ 광고 (300×250) - sticky
├─ 뉴스레터 구독 폼
├─ 인기 글 목록
└─ 광고 (300×600) - 선택사항
```

## 모바일 최적화 (핵심!)

트래픽의 60-70%는 모바일입니다. 모바일 광고를 별도로 최적화해야 합니다.

### 모바일 필수 배치

1. **앵커 광고 (하단 고정)**: 화면 하단에 고정되는 광고로 CTR이 매우 높습니다
2. **본문 상단**: 제목 바로 아래
3. **인콘텐츠**: 300-400자마다 1개 (최대 2개)

### 주의사항

- 모바일에서 광고가 콘텐츠를 가리면 구글 정책 위반
- 화면 전체를 덮는 팝업 광고 금지
- 광고와 콘텐츠 사이에 명확한 구분 필요

## A/B 테스트로 최적 배치 찾기

같은 광고라도 위치에 따라 CTR이 다릅니다. 직접 테스트해야 최적 배치를 찾을 수 있습니다.

### 테스트 방법

1. **기간**: 최소 2주, 동일 기간 비교
2. **변수**: 한 번에 하나만 변경 (위치 OR 크기)
3. **지표**: CTR, RPM, 총 수익 비교
4. **기록**: 구글 서치콘솔 + 애드센스 대시보드 병행

### 테스트 체크리스트

- [ ] 본문 상단 vs 도입부 다음
- [ ] 본문 중간 위치 (1/3 vs 1/2 vs 2/3)
- [ ] 사이드바 sticky 광고 유무
- [ ] 앵커(하단 고정) 광고 활성화 여부
- [ ] 반응형 vs 고정 크기

## 광고 색상 및 스타일 최적화

### 기본 원칙

- **배경**: 콘텐츠와 자연스럽게 어울리는 색상 (과도한 대비 피하기)
- **테두리**: 없거나 얇게 (광고임을 강조하지 않는 게 더 클릭률 높음)
- **라벨**: "광고" 표시는 구글 정책 의무, 숨기거나 작게 만들기

> ⚠️ 주의: "여기를 클릭하세요" 같은 광고 클릭 유도 문구는 정책 위반입니다.

## 자동 광고 vs 수동 광고

### 자동 광고 (Auto Ads)

구글이 최적 위치에 자동으로 광고를 삽입합니다.

**장점**: 설정 간편, AI가 지속 최적화
**단점**: 광고가 원치 않는 위치에 삽입될 수 있음, 디자인 방해 가능

### 수동 광고

직접 원하는 위치에 광고 코드를 삽입합니다.

**장점**: 완전한 제어권, 디자인 보호
**단점**: 지속적인 최적화 필요

**권장**: 수동 광고로 3-4개 핵심 위치를 관리하고, 자동 광고는 추가 보완으로 활용

## 수익 향상을 방해하는 실수 5가지

1. **광고 과밀 배치**: 화면에 광고가 너무 많으면 이탈률 상승
2. **모바일 최적화 무시**: PC만 최적화하고 모바일 방치
3. **테스트 없이 유지**: 한 번 설정 후 수년간 방치
4. **콘텐츠 중간 과도한 광고**: 읽기 흐름을 깨면 이탈
5. **정책 위반 광고**: 클릭 유도, 오해 유발 배치로 계정 정지 위험

## 실전 적용 순서

1. **현재 상태 기록**: 스크린샷 + 애드센스 RPM/CTR 수치 저장
2. **앵커 광고 활성화**: 즉시 CTR 향상 효과
3. **본문 상단 광고 추가**: 제목 바로 아래 배치
4. **2주 후 결과 비교**: 개선 여부 확인
5. **다음 위치 테스트**: 반복
