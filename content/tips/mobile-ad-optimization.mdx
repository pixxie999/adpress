## 핵심 요약

> 모바일 트래픽이 60-75%를 차지하지만, 대부분의 블로거는 PC 기준으로만 광고를 설정합니다. 모바일을 따로 최적화하면 수익이 즉시 상승합니다.

## 왜 모바일 최적화가 중요한가?

구글 애널리틱스 데이터 기준, 대부분의 한국 블로그는:

- **모바일 트래픽**: 65-75%
- **PC 트래픽**: 20-30%
- **태블릿**: 5% 이하

모바일 수익을 최적화하지 않으면 전체 수익의 70%를 방치하는 셈입니다.

## 3단계 최적화 전략

### 1단계: 앵커 광고 활성화 (즉시 효과)

앵커 광고는 화면 하단에 고정되는 광고로, 스크롤해도 사라지지 않습니다.

**설정 방법:**
1. 애드센스 로그인
2. 광고 → 사이트 단위 광고 → 자동 광고
3. "앵커 광고" 토글 켜기

**예상 효과**: CTR 0.5-1.5%p 상승, RPM 15-30% 향상

### 2단계: 모바일 전용 인콘텐츠 광고

본문 텍스트 사이에 320×100 또는 반응형 광고를 삽입합니다.

**최적 위치:**
- 제목 바로 아래 (도입부 전)
- 본문 중간 (500-700자 지점)
- 결론 직전

**주의사항:**
- 광고 간격은 최소 500자 이상 유지
- 한 화면에 광고가 2개 이상 보이지 않도록
- 폰트 크기 16px 이상으로 가독성 확보

### 3단계: 페이지 속도 개선

광고가 아무리 잘 배치되어도 페이지가 느리면 이탈합니다.

**PageSpeed 목표 점수:** 모바일 70 이상 (100점 만점)

**속도 개선 방법:**
- 이미지 WebP 형식 변환 (파일 크기 30-50% 감소)
- 이미지 lazy loading 적용
- 불필요한 스크립트 제거
- CDN 사용 (Cloudflare 무료)

**확인 방법:** pagespeed.web.dev에서 URL 입력

## 모바일 광고 설정 체크리스트

- [ ] 앵커 광고 활성화
- [ ] 반응형 광고 단위 사용 (고정 크기 대신)
- [ ] 본문 상단에 모바일 친화적 광고 1개
- [ ] 본문 중간에 인콘텐츠 광고 1-2개
- [ ] 모바일 페이지 속도 70점 이상
- [ ] 이미지 WebP 최적화 완료
- [ ] 글자 크기 16px 이상 (가독성)
- [ ] 터치 대상 간격 48px 이상

## 즉시 확인할 지표

모바일 최적화 후 2주 뒤 아래를 확인하세요:

- **애드센스**: 모바일 CTR, RPM 변화
- **애널리틱스**: 모바일 이탈률, 세션 시간
- **PageSpeed**: 모바일 점수
