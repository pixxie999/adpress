## 핵심 요약

애드센스 거절의 대부분은 **콘텐츠 품질 부족**과 **정책 위반**으로 발생합니다. 아래 7가지 사유를 확인하고 수정 후 재신청하세요.

## TOP 7 거절 사유

### 1. 가치 없는 콘텐츠 (가장 흔한 사유)

구글이 가장 많이 사용하는 거절 사유입니다. "콘텐츠가 충분하지 않거나 광고주에게 가치가 없다"는 의미입니다.

**해결법**:
- 각 글 1,000자 이상으로 작성
- 직접 경험과 데이터 기반 콘텐츠 작성
- 최소 15~20개 게시글 확보

### 2. 저작권 위반 콘텐츠

타 사이트에서 복붙한 텍스트, 무단 이미지 사용이 해당됩니다.

**해결법**:
- 모든 이미지를 직접 촬영하거나 저작권 무료 이미지 사용 (Unsplash, Pixabay)
- 인용 시 출처 명시

### 3. 필수 페이지 누락

개인정보처리방침, 이용약관, 소개 페이지가 없으면 거절됩니다.

**해결법**: 4가지 필수 페이지 추가
- 개인정보처리방침
- 이용약관
- 소개(About)
- 문의하기(Contact)

### 4. 트래픽 부족

사이트 방문자가 거의 없는 신규 사이트는 거절될 수 있습니다.

**해결법**: 최소 3개월 이상 운영 후 신청, 또는 SNS/커뮤니티로 초기 트래픽 확보

### 5. 성인/유해 콘텐츠

정책에 위반되는 콘텐츠가 포함된 경우 즉시 거절됩니다.

**해결법**: 콘텐츠 전수 검토 및 위반 게시글 삭제 또는 비공개 처리

### 6. 과도한 광고 코드 삽입

이미 다른 광고 코드가 너무 많으면 거절될 수 있습니다.

**해결법**: 신청 전 다른 광고 코드를 모두 제거

### 7. 정치/종교적 논란 콘텐츠

광고주가 꺼리는 민감한 주제의 콘텐츠가 많으면 거절됩니다.

**해결법**: 논란이 될 수 있는 게시글 비공개 처리 후 신청

## 재신청 체크리스트

- [ ] 게시글 20개 이상, 각 1,000자 이상
- [ ] 모든 이미지 저작권 확인
- [ ] 필수 4개 페이지 구성 완료
- [ ] 다른 광고 코드 전체 제거
- [ ] 유해/위반 콘텐츠 없음 확인
- [ ] 모바일 반응형 디자인 확인
