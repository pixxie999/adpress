## 핵심 요약

> 애드포스트는 네이버 블로그 운영 기간과 콘텐츠 품질이 핵심입니다. 아래 조건을 갖추고 신청하면 1-2주 내 승인 가능합니다.

## 애드포스트 승인 필수 조건

### 기본 조건 (필수)

| 항목 | 기준 |
|------|------|
| 블로그 운영 기간 | 최소 90일 이상 |
| 게시글 수 | 최소 50개 이상 |
| 이웃 수 | 공식 기준 없음 (많을수록 유리) |
| 주제 | 일관된 카테고리 (잡블로그 불리) |

### 콘텐츠 품질 조건

- 게시글당 최소 **500자 이상** (1,000자 이상 권장)
- 직접 촬영한 **사진 3장 이상** 포함
- **복붙, 스크랩** 글 없어야 함
- 최근 30일 내 **10개 이상** 신규 게시글

## 신청 전 준비 체크리스트

- [ ] 90일 이상 꾸준히 운영 중
- [ ] 총 게시글 50개 이상
- [ ] 최근 한 달 게시글 10개 이상
- [ ] 게시글 평균 글자 수 700자 이상
- [ ] 직접 촬영 이미지 포함 비율 70% 이상
- [ ] 명확한 블로그 주제 (카테고리 3개 이내 집중)
- [ ] 불법/성인 콘텐츠 없음
- [ ] 저작권 침해 콘텐츠 없음

## 승인 통과를 높이는 실전 팁

### 1. 신청 직전 2주가 중요

심사관이 가장 최근 글을 집중적으로 봅니다. 신청 전 2주 동안 질 높은 글을 집중 발행하세요.

### 2. 대표 카테고리 3개 이내로 정리

"먹방 + 여행 + 맛집 + 뷰티 + IT" 잡블로그는 불리합니다. 2-3개 카테고리에 집중하세요.

### 3. 시리즈 포스팅 발행

같은 주제를 시리즈로 발행하면 전문성이 돋보입니다.
예: "애드센스 완전 정복 1편 ~ 5편"

### 4. 거절 후 재신청 전략

거절되었다면 최소 **30일 후** 재신청하세요. 거절 사유를 정확히 파악하고 보완한 후 신청해야 재거절을 막을 수 있습니다.

## 애드포스트 신청 방법

1. 네이버 애드포스트 사이트 접속 (adpost.naver.com)
2. 네이버 로그인 후 "광고 신청"
3. 블로그 URL 입력
4. 심사 대기 (평균 1-2주)
