## RPM이란?

RPM(Revenue Per Mille)은 페이지뷰 1,000회당 평균 수익입니다. RPM이 높을수록 같은 트래픽으로 더 많은 돈을 벌 수 있습니다.

- **한국 평균 RPM**: 1,500~3,000원
- **최적화 후 RPM**: 3,000~6,000원

## 5가지 RPM 상승 전략

### 1. 고CPC 키워드 중심 콘텐츠 작성

RPM은 CPC(클릭당 단가)와 직결됩니다. 고CPC 키워드 카테고리:

- 금융/보험/대출: 500~2,000원/클릭
- 법률/세무: 400~1,500원/클릭
- 부동산/인테리어: 200~800원/클릭
- IT/SaaS 서비스: 200~600원/클릭

### 2. 광고 자동 최적화 활성화

구글 애드센스의 자동 광고 기능을 켜두면 구글이 최적 위치에 광고를 배치합니다. 처음 2~4주는 테스트 기간이니 성급하게 끄지 마세요.

### 3. 페이지당 광고 개수 최적화

너무 적거나 너무 많으면 RPM이 떨어집니다. 권장 배치:
- 본문 상단 (접히기 전)
- 본문 중간 (2,000자 기준 절반 지점)
- 본문 하단

### 4. 모바일 광고 최적화

한국 트래픽의 70% 이상이 모바일입니다. 모바일 광고 최적화가 RPM에 직접적인 영향을 줍니다:
- 반응형 광고 단위 사용
- 모바일에서 광고가 잘리지 않는지 확인

### 5. 콘텐츠 길이와 체류 시간 증가

긴 글일수록 스크롤 중 광고 노출 횟수가 늘어납니다:
- 목표: 게시글당 3분 이상 체류
- 방법: 영상, 인포그래픽, 표 활용

## 즉시 적용 체크리스트

- [ ] 현재 RPM 애드센스 대시보드에서 확인
- [ ] 자동 광고 설정 켜기
- [ ] 반응형 광고 단위로 교체
- [ ] 고CPC 키워드 1개 이상 글 작성
- [ ] 모바일에서 광고 미리보기 확인
