# 사용 가이드
# ============================================

# 1. 우선순위별 생성 순서:
#    - high: 즉시 생성 (사이트 런칭 후 1주 내)
#    - medium: 2주 내 생성
#    - low: 1개월 내 생성

# 2. 카테고리별 비율 유지:
#    - guides: 25% (심층 가이드)
#    - tips: 50% (빠른 팁)
#    - reviews: 20% (비교/리뷰)
#    - income-reports: 5% (월간 리포트)

# 3. AI 생성 시 프롬프트 포함 사항:
#    - 검색 의도 파악 (정보형/거래형/탐색형)
#    - 경쟁 콘텐츠 분석
#    - 실전 경험 느낌 강조
#    - 데이터/수치 근거 포함

# 4. 내부 링크 전략:
#    - 허브(guides) ← → 스포크(tips) 양방향 연결
#    - 관련 키워드끼리 그룹화
#    - 예: "애드센스 승인" 가이드 → "승인 거절 사유" 팁

# 5. 주간 발행 계획 (예시):
#    - 1주차: high 우선순위 5개
#    - 2주차: high 우선순위 5개 + medium 2개
#    - 3주차: medium 우선순위 5개
#    - 4주차: medium 우선순위 5개 + low 2개
#    - 이후: low 우선순위 + 추가 키워드
